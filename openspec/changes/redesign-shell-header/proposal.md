# Proposal: Redesign Shell Header

## Summary

Refine the global shell header into a clearer two-layer structure so the desktop guide app has a stable navigation center and a lightweight section context bar.

## Scope

- simplify the global header into brand, navigation, and actions zones
- move section context out of the brand block into a dedicated context bar
- keep existing page content modules intact while reducing top-level visual noise
- preserve responsive behavior for narrower desktop widths

## Out of Scope

- redesigning feature page content layouts
- adding new search capabilities or search results pages
- changing routing structure or section ownership
