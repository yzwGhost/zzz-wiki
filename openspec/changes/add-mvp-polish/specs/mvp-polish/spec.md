# MVP Polish

## ADDED Requirements

### Requirement: Centralized sync management entry

The application SHALL keep sync execution management centralized in the sync center instead of duplicating full execution controls across multiple pages.

#### Scenario: View sync access from home page

- **WHEN** the user visits the home page
- **THEN** the page links to the sync center
- **AND** does not duplicate the full sync execution and result management panel

### Requirement: Current MVP status documentation

The repository SHALL include a document that states the current MVP scope, completed items, pending items, and current startup commands.

#### Scenario: Read MVP status

- **WHEN** a developer opens the MVP status document
- **THEN** they can see completed modules, missing capabilities, and validation guidance
