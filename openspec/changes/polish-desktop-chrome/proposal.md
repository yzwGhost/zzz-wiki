# Proposal: Polish Desktop Chrome

## Summary

Align the desktop window chrome and global scrollbar styling with the current dark project theme.

## Scope

- remove the default Electron application menu bar from the desktop window
- theme the native top chrome to better match the app shell
- add project-consistent scrollbar styling for the renderer

## Out of Scope

- building a fully custom draggable title bar in the renderer
- changing window controls behavior
- redesigning page-level layout or content modules
