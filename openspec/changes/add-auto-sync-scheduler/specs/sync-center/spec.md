## MODIFIED Requirements

### Requirement: Sync center management page
The desktop application SHALL provide a dedicated sync center page for manual sync execution and status visibility.

#### Scenario: View and manage auto sync
- **WHEN** the user opens the sync center page
- **THEN** the page shows whether auto sync is enabled, the current interval, the next expected run time, and the latest automatic sync result
- **AND** the page allows the user to update the enabled state and interval without leaving the sync center
